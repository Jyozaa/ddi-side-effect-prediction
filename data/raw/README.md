# Dataset Download

The dataset files are too large to store directly in this GitHub repository.

Instead, the compressed `data/` folder has been uploaded to Google Drive. You can download it here:

https://drive.google.com/file/d/1dg7CLVzi9XxiUQ2TeM7muRdfv6xxSGMb/view?usp=sharing

Original source datasets are also available from:

- TwoSIDES: https://tatonettilab-resources.s3.amazonaws.com/nsides/TWOSIDES.csv.gz
- DGIdb: https://dgidb.org/downloads

After downloading, unzip the archive into the project root so the folder structure is:

```text
data/
├── raw/
└── processed/
```
